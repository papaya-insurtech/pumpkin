# VS Code Recommended Extensions

Clone this repo. Open the extensions tab in VS Code (macOS: Shift+Cmd+X, Windows: Ctrl+Shift+X), these extensions are listed in the recommendation section.

## AI & Coding Assistance

1. Claude Code: Install via CLI (`curl -fsSL https://claude.ai/install.sh | bash`). Works alongside VS Code / Cursor as a terminal tool.
2. GitHub Copilot: [GitHub.copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot). Papaya sponsors all developers.
3. Kiro: [Kiro](https://kiro.dev/). The extension is automatically installed when you install Kiro.
