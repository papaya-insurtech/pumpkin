# Onboarding Guide

Welcome to the Papaya engineering team. This guide covers the essential steps to get you up to speed with our AI-agent-first workflow and development practices.

## Week 1: Setup & Foundations

### AI Tools Setup

Start here. AI tools are central to all work at Papaya.

1. Follow the [Claude Setup Guide](./resources/Claude_Setup.md) to install Claude Code, Claude Desktop, and team convention skills.
2. Read [AI Agent Philosophy](../Software_Artisan_Handbook/resources/AI_Agent_Workflow/AI_Agent_Philosophy.md) to understand our three-tool model.
3. Read [Claude Code Best Practices](../Software_Artisan_Handbook/resources/AI_Agent_Workflow/Claude_Code_Best_Practices.md) for effective usage patterns.

### Handbook & Process

4. Read the [Software Artisan Handbook](../Software_Artisan_Handbook/Software_Artisan_Handbook.md) -- the table of contents links to all key documents.
5. Read the [Development Process](../Software_Artisan_Handbook/resources/Development_Process/Development_Process.md) and [Task Lifecycle](../Software_Artisan_Handbook/resources/Development_Process/resources/Task_Lifecycle.md).
6. Read the [Git Workflow](../Software_Artisan_Handbook/resources/Development_Process/resources/Git_Workflow.md).

### Domain Knowledge

7. Read [Medical Insurance 101](../Software_Artisan_Handbook/resources/Medical_Insurance_101.md). This domain knowledge is essential for understanding the codebase and providing correct context to AI agents.

## Week 2: Practice & Integration

### Agent Practice

8. Complete a small task end-to-end using Claude Code: write code, run tests, create a PR.
9. Use Claude Chat to research a concept you're unfamiliar with and summarize your findings.
10. Use Claude Cowork to draft a test plan or document.

### Code Review

11. Read the [Code Review](../Software_Artisan_Handbook/resources/Development_Process/resources/Code_Review/Code_Review.md) process.
12. Read [Reviewing AI Generated Code](../Software_Artisan_Handbook/resources/Development_Process/resources/Code_Review/resources/Reviewing_AI_Generated_Code.md) -- understand what to watch for.
13. Review a teammate's PR to practice the review process.
